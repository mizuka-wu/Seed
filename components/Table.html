# 参数

# seeds

  • 类型: Seed[]
  • 必填: 是

基本属性，根据Seed配置生成整个列表

# rowKey

  • 类型: string
  • 默认值: _key

会根据这个属性绑定每一行的 key，如果没有的话，在获取fetchList的之后会为每一行生成一个uuid
